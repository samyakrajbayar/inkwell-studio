# ✒️ Inkwell Studio

A modern full-stack application designed to provide a seamless writing and content creation experience. Inkwell Studio combines a clean web interface with a powerful backend to help users create, manage, and organize content efficiently.

---

## 🚀 Features

* 📝 **Rich Writing Interface** – Clean and distraction-free UI for content creation
* ⚡ **Full-Stack Architecture** – Integrated frontend and backend system
* 🗄️ **Data Persistence** – Store and manage content reliably
* 🌐 **Web-Based Access** – Accessible through the browser
* 🔧 **Developer-Friendly Setup** – Easy to run and extend locally

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/samyakrajbayar/inkwell-studio.git
cd inkwell-studio
```

### 2. Setup backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3. Setup frontend

```bash
cd frontend
npm install
npm start
```

### 4. Open the app

```
http://localhost:3000
```
